Manual to use the Website

Welcome to Comic Vault Raiders! This website is designed to help you catalog your comic book collection and share recommendations with your friends. Here’s a quick guide on how to use the site.

Getting Started

To begin, create an account by clicking the “Register” button on the main page. Once your account is created, you can log in and start adding comic books to your collection.

Navigation

At the top of the page, you will see a navigation bar with links to the main sections of the website. You can move between these sections by clicking the links.
If you click the triangle on the side of the navigation bar, you can switch its position to the left or right side of the screen.

My Comics Page

After logging in, navigate to the “My Comics” page to view and manage your collection.
In the first section, you will see the comics already in your collection. This section displays comics that have a cover image in the system. You can remove comics from your collection using the “X” button.

Below this, you will find a filterable grid that represents your full collection of comics added to your account.

The last section is the Comic Management component. Here, you can search for comics in the system and add them either to your collection or to your wishlist. You can search by title (keywords are enough) or by author.
This component also includes an “Add New Comic” button at the bottom. This allows you to add a new comic to the system. A scan mode is available to help pre-fill data using the comic’s ISBN. The form contains both required and optional fields. It is recommended to fill in as much information as possible. However, the system runs a frequent background process that checks newly added comics and attempts to complete missing data automatically. If you provide only the title and author, the system will try to retrieve the remaining details for you. This process is not perfect, but it can save time. Once the comic is added to the system, you can choose to add it to your collection or your wishlist.

Wishlist Page

On the “Wishlist” page, you can view the comics you have added to your wishlist. You can remove comics using the “X” button. The structure and logic of this page are similar to the “My Comics” page, but it only displays comics from your wishlist.

Settings Page

If you are a regular user, this section is currently a work in progress. Please check back later.

If you are an administrator, you will see the Comic Scraper component here. This tool allows you to run the comic scraper process manually. The scraper collects comic book data from various sources and adds it to the system. You can enter keywords that the scraper will use to search for comics. After clicking the “Run Scrape” button, the keywords are added to a queue and processed in the background. You can continue using the website while the scraper is running. Once the data is ready, it will be added to the system automatically. If a keyword is already in the queue, it will not be added again. The scraper runs every 5 minutes. If the daily quota is reached, the process pauses and resumes the next day.
